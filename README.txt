1、paper_1文件夹下为最小点覆盖....算法的仿真程序。
   paper_2文件夹下为以时延为约束条件的部署算法的仿真程序。
   paper_4文件夹下为以生存性为约束条件的部署算法的仿真程序。

2、每个算法仿真程序文件夹中，只需运行main.py文件就可以得出部署方案。

3、在画对比图之前，要将三种算法在同一参数下运行一遍。

4、程序中的parameter.py文件可以控制三种算法中的拓扑参数。

5、draw开头的py文件为画结果比较图的文件，其中有些程序中的show语句注释掉了，运行之后可能不会显示图片。（注：show语句不能放在savefig语句之前，保存图片时会出现错误）

6、networkx文件夹下为图形化仿真演示程序，包括部署对比图和算法流程演示。
（注：程序中的暂停语句不能用time.sleep代替plt.pause，否则动画显示时会出现错误）

7、网上基本没有关于networkx中动态变化拓扑的资料，在写相关程序时，可以参考该程序中的代码逻辑。


8、所有程序在Python3中即可运行，运行前请装好相应的库。
